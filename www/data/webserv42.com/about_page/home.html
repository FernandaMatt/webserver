About The Project

The webserver project is about building a functional HTTP server from scratch. The server should be able to serv a fully static website and handle CGI. To accomplish that we needed a deep understanding of network protocols and concurrency.

The server needs to handle HTTP requests and responses, manage connections, and ensure server stability under different conditions.

The server connections needs to be non-blocking through the use of the select, poll, or epoll system calls. In this project we used epoll!
